# Code of Conduct

We want Udaan to be a welcoming and respectful project for everyone.

Expected behavior
- Be respectful and considerate in communication.
- Use constructive language when giving feedback.
- Respect privacy and personal boundaries.

Unacceptable behavior
- Harassment, hateful or discriminatory language.
- Personal attacks, doxxing, or threats.
- Repeated disruptive behavior after a warning.
